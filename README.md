# A2_Group14A_MidTermGame

# Project Title

Panic Mode

# Group Number

Group 14A

# Description

This project is a p5.js game prototype designed to help players understand the experience of **panic disorder** through gameplay mechanics. The game simulates increasing cognitive load, time pressure, and environmental triggers that mirror the overwhelming sensations often associated with panic attacks.

Players must navigate obstacles while managing stress levels. As the game progresses, tasks and stimuli increase, representing how panic disorder can escalate under pressure. A grounding mechanic allows the player to briefly stabilize themselves, reflecting real-world coping strategies used during panic attacks.

The goal of the game is to create an **entertainment-focused** experience that also encourages empathy by allowing players to experience escalating stress and the challenge of maintaining control under pressure.

# Setup and Interaction Instructions

## Controls

- **Move Left:** A or ←
- **Move Right:** D or →
- **Jump:** Space

## Gameplay

- Avoid obstacles and navigate the level.
- As the game progresses, stress and distractions increase.
- Use the grounding mechanic to temporarily stabilize when stress becomes overwhelming.

## Goal

Reach the end of the level while managing rising pressure and maintaining control of the character.

# Iteration Notes

## Post-Playtest

Based on peer playtesting, we made the following design changes:

1. **Simplified the instruction screen** because testers reported that the original instructions were too long and difficult to quickly understand.
2. **Adjusted the early level difficulty** so the first level introduces mechanics gradually instead of overwhelming players immediately.
3. **Improved clarity of the game objective** so players can more easily understand what they are supposed to do during gameplay.

These changes were implemented to improve onboarding and make the core mechanics easier to learn before introducing more complex challenges.

## Post-Showcase

During the mid-term showcase, instructors and TAs suggested several improvements that we plan to implement:

1. **Redesign the difficulty progression** using research on panic disorder triggers so that stress increases gradually as the level progresses.
2. **Improve the grounding mechanic** by simplifying it so players can activate it easily rather than turning it into a stressful mini-game.
3. **Reduce text and add icons to the instruction screen** so key controls and mechanics are clearer and easier to understand.

These improvements aim to better reflect the psychological experience of panic disorder while keeping the game engaging and accessible.
